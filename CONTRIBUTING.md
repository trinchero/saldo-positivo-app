# Contributing to Saldo+

First off, thank you for considering contributing to Saldo+! It's people like you that make Saldo+ such a great tool.

## Code of Conduct

This project and everyone participating in it is governed by a Code of Conduct. By participating, you are expected to uphold it. Please report unacceptable behavior to andrea_trinchero@icloud.com.

## How to Contribute

### Reporting Bugs

Before opening a bug report:

- Check if you can reproduce the problem in the latest version of Saldo+.
- Provide clear steps to reproduce.
- Include screenshots or screen recordings when possible.

### Suggesting Enhancements

When proposing a feature:

- Explain the user problem youâ€™re solving.
- Provide examples of where in Saldo+ the improvement would apply.
- If possible, attach mockups or UI notes.

### Pull Requests

1. Fork the repository.
2. Create your feature branch (`git checkout -b feature/awesome-change`).
3. Commit your changes (`git commit -m 'Describe your change'`).
4. Push to the branch (`git push origin feature/awesome-change`).
5. Open a Pull Request.
